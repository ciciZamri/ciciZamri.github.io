Limits

A limit is the value that a function approaches as the input approaches some value

\(\lim_{x\to c} f(x) = L\)
The limit of the function \(f\) as \(x\) approaches to \(c\) is \(L\).

Power rule

Power rule is used to find the derivative of an expression in the form \(x^n\)

\(\frac{d}{dx} x^n = nx^{n-1}\)

Properties

  • \(\frac{d}{dx} c = 0\)

  • \(\frac{d}{dx} c \cdot f(x) = c \cdot \frac{d}{dx}f(x)\)

  • \(\frac{d}{dx} [f(x) + g(x)]= \frac{d}{dx}f(x) + \frac{d}{dx}g(x)\)

  • \(\frac{d}{dx} [f(x) - g(x)]= \frac{d}{dx}f(x) - \frac{d}{dx}g(x)\)

  • \(\frac{d}{dx} x = 1\)

  • \(\frac{d}{dx} x^n = nx^{n-1}\)
